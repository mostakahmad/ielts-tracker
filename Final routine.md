# IELTS Academic Band 7+ Complete 45-Day Detailed Routine

## Profile

* Target: Overall Band 7+
* Exam Type: Academic IELTS
* Current Level: Beginner
* Daily Study Time: 4 hours
* Morning: 2 hours
* Evening/Night: 2 hours
* Weak Areas: Listening, Reading, Writing, Speaking, Vocabulary, Grammar
* Study Mode: Self-study

## Core Daily Rule

Every day must include:

1. Listening
2. Reading
3. Writing
4. Speaking
5. Vocabulary
6. Grammar
7. Error correction

Practice without correction is not enough. Every wrong answer must be reviewed.

---

# Daily Time Structure

## Morning Session — 2 Hours

| Time   | Module     | Task                                                     |
| ------ | ---------- | -------------------------------------------------------- |
| 20 min | Vocabulary | Learn 18–20 words/collocations from the daily topic list |
| 40 min | Listening  | Practice one IELTS Listening section or focused skill    |
| 45 min | Reading    | Practice one Reading question type or passage            |
| 15 min | Review     | Correct mistakes and update error notebook               |

## Evening/Night Session — 2 Hours

| Time   | Module   | Task                                                     |
| ------ | -------- | -------------------------------------------------------- |
| 45 min | Writing  | Task 1 or Task 2 focused practice                        |
| 30 min | Speaking | Speaking Part 1, 2, or 3 practice with recording         |
| 25 min | Grammar  | One grammar topic + sentence correction                  |
| 20 min | Review   | Daily reflection, error correction, next-day preparation |

---

# Weekly Structure

| Week               | Focus                         | Main Target                                             |
| ------------------ | ----------------------------- | ------------------------------------------------------- |
| Week 1, Days 1–7   | IELTS foundation              | Understand format, build basic English control          |
| Week 2, Days 8–14  | Core skill building           | Learn major question types and basic writing structures |
| Week 3, Days 15–21 | Question-type mastery         | Practice Reading/Listening question types deeply        |
| Week 4, Days 22–28 | Band 6 to Band 7 transition   | Timed practice, essay improvement, speaking fluency     |
| Week 5, Days 29–35 | Mock test and weakness repair | Full tests, targeted correction, confidence building    |
| Week 6, Days 36–45 | Final Band 7 polishing        | Full mock tests, final revision, error elimination      |

# Band 7+ Final Strategy

## Listening

Target 30+ out of 40. Section 1 should be almost perfect. Section 3 and 4 require transcript review.

## Reading

Target 30+ out of 40. Do not read every word. Use question strategy, synonyms, and time control.

## Writing

Task 1 must have a clear overview. Task 2 must have a clear position, logical paragraphs, accurate grammar, and natural vocabulary.

## Speaking

Speak naturally. Do not memorize full answers. Use answer + reason + example + detail.

## Vocabulary

Use active vocabulary daily. Knowing words is not enough. You must use them correctly.

## Grammar

Band 7 does not mean perfect grammar, but errors should not frequently block meaning. Accuracy is more important than using very complex sentences.
